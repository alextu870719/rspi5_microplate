# 384 Column Sequential CSV File

這個CSV文件演示384孔板按列依序點亮的範例。

## 檔案內容：384_Column_Sequential.csv

- **總步驟數**：24步（對應24列）
- **每步驟孔數**：16個孔（對應A-P行）
- **點亮模式**：每步點亮一整列

## 步驟說明：

| 步驟 | 點亮的列 | 孔位範圍 |
|------|----------|----------|
| 1 | Column 1 | A01-P01 |
| 2 | Column 2 | A02-P02 |
| 3 | Column 3 | A03-P03 |
| ... | ... | ... |
| 24 | Column 24 | A24-P24 |

## 使用方式：

1. 在程式中點擊「Load CSV」
2. 選擇此檔案（384_Column_Sequential.csv）
3. 使用「Next/Previous」按鈕瀏覽每個步驟
4. 每個步驟會點亮該列的所有16個孔
